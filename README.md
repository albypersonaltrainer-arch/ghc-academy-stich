# GHC Academy

Primera fase del proyecto: frontend premium preparado para desplegar en Vercel.

## Páginas incluidas

- `/` Página principal
- `/cursos` Catálogo de cursos
- `/alumno` Portal visual del alumno
- `/admin` Panel visual privado del administrador

## Instalación local

```bash
npm install
npm run dev
```

Abrir en el navegador:

```bash
http://localhost:3000
```

## Deploy en Vercel

1. Subir este proyecto completo a GitHub.
2. Entrar en Vercel.
3. Importar el repositorio.
4. Deploy.

## Nota importante

Esta versión es la Fase 1: diseño frontend premium con datos mock.
Todavía no incluye backend real, pagos reales, login real ni base de datos real.
